# Discord Datamining

이 레포지토리는 **Discord** 의 클라이언트 및 API의 변경 사항을 자동으로 추적하고 기록하기 위해 생성되었습니다. 프로그램이 주기적으로 실행되어 실험(Experiments), 내부 문자열(Strings), API 엔드포인트 등의 변화를 감지하고, 변경 이력을 이 레포지토리에 자동으로 커밋 및 푸시합니다.


## ⚙️ 작동 방식

1.  데이터 수집기가 주기적으로 **Discord** 의 최신 빌드 데이터를 가져옵니다.
2.  **이전 커밋** 의 데이터와 비교하여 변경점을 추출합니다.
3.  **변경 사항** 이 감지되면 로컬 파일 시스템을 업데이트합니다.
4.  **자동화 스크립트** 가 Git 명령어를 실행하여 이 레포지토리에 변경된 파일들을 반영합니다.


## 주요 기능

- Discord Canary의 새 웹 클라이언트 빌드 확인
- 현재 Canary 빌드의 JavaScript chunk 다운로드
- 실험 메타데이터 추출
- API 엔드포인트 상수 및 템플릿 추출
- 영어 문자열 후보 추출
- 최신 변경 사항과 누적 이력을 `data/`에 저장
- 새 빌드가 감지되면 데이터 변경 사항을 커밋하고 푸시
